Online Quiz Game

Online quiz game is an interactive game, containing a series of questions and riddles that should be answered in a certain amount of time. The player’s goal is to answer all the questions as quickly and accurately as possible.

Instructions

● At each round, a new question will be presented
● Each question contains multiple possible answers, only one is correct
● Each play has limited time to respond to each question
● There are two types of questions: True/False or Multiple Choice
● True/False Questions has a time limit of 15 seconds to answer
● Multiple Choice Questions has a time limit of 30 seconds to answer
● No score will be granted if a question hand't been answered
● Each correct answer is scored

Score Calucation
The total score is calculated for each question by the following formula:
QUESTION_DIFFICULTY * (QUESTION_TOTAL_TIME - TIME_TOOK_TO_ANSWER)

● When QUESTION_DIFFICULTY is "Easy" the multiplier is 1
● When QUESTION_DIFFICULTY is "Medium" the multiplier is 5
● When QUESTION_DIFFICULTY is "Hard" the multiplier is 10

Tips

● Tips can only be used when playing a Multiple Choice game
● Each tip can only be used once per game, so be careful!
● 50/50 Elimination - 10 points
● Show the first letter of the answer - 20 points
● Show an image related to the answer - 15 points

Leaderboard

● A table for showing the 50 highest scores recorded on your browser

Notes

● The questions are generated from the free API: Open Trivia Database
● The images are fetched with Google API Custom Search
● The images API has a rate limit of 100 queries a day
● Images that had been searched are cached within the storage of the program
